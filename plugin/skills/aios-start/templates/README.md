# templates

The actual brain templates live in `/plugin/templates/`, one level up from the `aios-start` skill.

This directory is intentionally blank. It exists as a placeholder in case the `aios-start` skill ever needs skill-local templates (e.g. onboarding-only snippets) distinct from the main brain templates rendered during install.

During install, the skill reads from `../../templates/` — not from this folder.
