# Go usage-based, not seat-based, for v1

## Claim
v1 pricing is usage-based (metered by runs) with a generous free tier. No seats, no per-user pricing.

## Context
Seat-based pricing punishes the use case we want to encourage — a solo operator using the tool constantly. Usage-based rewards the loyal user and lowers the barrier to entry for the curious one.

## Alternatives considered
- Pure seat-based ($29/seat/month). Rejected — misaligned with our ICP.
- Hybrid (seat + usage). Rejected — confusing to explain, hard to budget against.

## Rationale
We want the pricing to match how the product creates value. Value is created per run, not per chair in a conference room.
